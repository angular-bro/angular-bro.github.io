Angular Bro

Angular Bro is an app written on top of Broccoli JS (bro) intended for large angular projects.
  • Angular JS Angular JS is the javascripting framework. Without it, this project would cease to exist.
  • Angular UI Router (included by default, but optional) Angular UI Router is the defacto router for Angular projects. However, apart from scaffolding, there is not a direct dependency on this.
  • Broccoli JS Broccoli JS is the asset pipeline. This is the backbone of the build process, where all of your transpiling, minification, linint, and unit testing happens.
  • Yeoman Yeoman is the scaffolding for the application. It has all the angular module templates and the logic to place them where you want.
  • Grunt (included by default, but optional) Grunt is an easy wrapper around broccoli and express for a mock server. Since this is only wrapping the build process, you can swap it out for whatever task runner you would prefer.

Prerequisites

This project relies on yeoman for code scaffolding and bower for dependency management.
  • Node >= 5.0.0
npm install -g yo bower broccoli grunt-cli

Installation Instructions

npm install -g generator-angular-bro

Getting Started

Generate a brand new application 
yo angular-bro [project-name]
├── app/
│   ├── app.js
│   ├── router.js
│   └── project-name.js
├── assets/
├── config/
│   └── environment.js
├── styles/
│   └── app.less
├── tests/
│   ├── e2e/
│   ├── helpers/
│   └── unit/
├── .jscsrc

├── bower.json

├── circle.yml
├── Gruntfile.js
├── index.html

View your new app 
grunt serve

Adding to your Application

Generate a new state 
yo angular-bro:state new-state
├── app/
│   ├── new-state/
│   │   ├── controller.js
│   │   ├── module.js
│   │   ├── state.js
│   │   └── template.html
Add your state to the router.js 
  import angular from 'angular';
  import config from 'config/environment';
+ import NewStateModule from 'app/new-state/module';

  angular.module('router', [
      'ui.router',
+     NewStateModule.name,
  ])
  .config(function ($locationProvider, $urlRouterProvider) {
      "ngInject";
      "use strict";

      $locationProvider.html5Mode(config.html5Mode);
      $urlRouterProvider.otherwise('/');
  });
